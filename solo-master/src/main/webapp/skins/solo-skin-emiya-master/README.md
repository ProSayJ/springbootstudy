## 安装方式

### 直接下载方式

在本页面下载。点击"Download ZIP"下载，将主题文件夹改名为emiya，放入solo的skins目录下即可在管理后台看到此主题。

### git clone方式

在solo的skins目录下使用命令即可安装皮肤到本地
```shell
git clone https://github.com/zjhch123/solo-skin-emiya emiya
```
